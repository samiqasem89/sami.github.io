# TechPulse
A responsive static frontend website for technology events and industry news. The design uses the original dark laptop hero image as a base and rebuilds the project into a complete Apple-inspired product-style landing page.

## Features

- Apple-inspired premium layout with fixed glass navigation
- Full-screen hero section using the provided header image
- Event cards with category filters
- Search across events, topics, locations, and companies
- News section with modal brief interaction
- Newsletter form with client-side confirmation
- Scroll reveal animations
- Responsive mobile navigation

## Files

- `index.html` — page structure and content
- `style.css` — responsive styling and layout
- `script.js` — filtering, search, modal, menu, form, and animation behavior
- `assets/header.jpg` — hero background image

## How to run

Open `https://samiqasem89.github.io/tech-events-news/` in a browser, or deploy the folder to GitHub Pages, Netlify, Vercel, or any static hosting provider.
